使用提醒

这个静态版不会保存历史数据。刷新页面、关闭浏览器或清空输入后，内容可能丢失。 整理完成后请立即导出 Markdown 或通过浏览器保存 PDF。需要历史记录、附件长期保存和多次查看，请使用完整前后端版本。

支持 txt / Markdown / docx / pdf / 图片。docx 和 pdf 解析使用浏览器端库；如果完全离线打开，txt/Markdown 和粘贴文本最稳定。

    $("files").addEventListener("change", async (event) => {
      const files = Array.from(event.target.files || []);
      for (const file of files) {
        if (file.type.startsWith("image/")) {
          const url = await readFileAsDataUrl(file);
          state.images.push(url);
          continue;
        }
        const text = await extractText(file);
        if (text) {
          $("rawText").value = [$("rawText").value, `## ${file.name}\n${text}`].filter(Boolean).join("\n\n");
        }
      }
      $("files").value = "";
      if (state.images.length) {
        $("imageSection").hidden = false;
        $("imageList").innerHTML = state.images.map((url) => ``).join("");
      }
    });
